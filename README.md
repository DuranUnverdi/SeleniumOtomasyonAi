# Selenium Test Automation Project

Bu proje, **Java** programlama dili ve **Selenium WebDriver** framework'ü kullanılarak geliştirilmiş bir test otomasyon projesidir. **Page Object Model (POM)** tasarım deseni ile yapılandırılmıştır.

## Kullanılan Teknolojiler
- Java 11+
- Selenium WebDriver 4.15.0
- TestNG 7.8.0
- Maven
- WebDriverManager

## Proje Hakkında
Proje, https://www.saucedemo.com/ web sitesi üzerinde otomatik test senaryoları çalıştırmak için tasarlanmıştır. Login işlemleri, kullanıcı doğrulama ve hata kontrolleri gibi temel test senaryolarını içerir.

Test senaryoları **Claude AI** yardımıyla geliştirilmiş ve optimize edilmiştir.

## Kurulum ve Çalıştırma
```bash
mvn clean install
mvn test
```
